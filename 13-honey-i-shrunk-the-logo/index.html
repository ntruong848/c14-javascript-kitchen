You may want to react to scroll changes on your page to change the size and position of elements

A simple way to do this is to listen to the "scroll" event on the document and use the window.scrollY property to decide what to do.

To shrink a logo, you can add a style to the classlist of your logo element that changes the height.

Put these concepts together to shrink the logo whenever the user scrolls more than 50 pixels into the page.
