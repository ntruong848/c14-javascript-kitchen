Creating elements with absolute position

We can use the same "background" technique as in parallax scroll to create an background element (div) that only exists to hold other absolutely positioned elements

We can set any "style" we want on a created element by setting css properties on the element.style.propertyName

Combine these techniques with to add an image (img) to the "background" div when the button is clicked. Use style.position="absolute" and style.top and style.left to place the new image somewhere in the background

Bonus points: randomly place the star, note that you can get the size of the background element from the .clientWidth and .clientHeight properties of the element.
