Adding to the document

You can create new elements in Javascript using the "createElement" function on the "document" object.

Note that the element is not laid out by the browser or until it is "append"ed to an existing element in the document.

When the button is clicked, create a "p" element with a message in it (assign the ".textContent" field of the created element). Append your new paragraph to the "outlet" div.
